GOOD
02 · Strengths

The Good

Acclime has invested meaningfully in foundational security. These controls are genuinely effective and represent real risk reduction — particularly against credential attacks and endpoint compromise.

Implemented · Critical
MFA Enforced Group-Wide
MFA required across VPN, admin accounts, email (O365), backup console and all cloud access. Legacy protocols blocked. Remote VPN restricted by IP in HK, China, Australia & Crowe. Strong protection against credential stuffing attacks.
Implemented · Critical
EDR + MDM Deployed
LMNTRIX + Microsoft Defender across all endpoints. Scripting blocked for standard users. Local admin rights removed. BitLocker encryption via Intune. Microsoft E3/E5 MDM for mobile devices. Solid endpoint posture.
Implemented · Critical
Multi-Layer Email Security
Microsoft Defender ATP + Perception Point running in tandem. KnowBe4 phishing simulation launching 1 April 2026 across Mauritius, Seychelles & UAE. Phishing is the #1 attack vector — this is the right investment.
Implemented · High
Immutable Backup & SIEM Logging
Immutable backups stored in Acronis. LMNTRIX Grid SIEM retains logs 3 months online + 9 months offline. Sources: Sonicwall, AWS, Entra ID, Defender. Forensic capacity exists if an incident occurs.
Implemented · High
Cloud Security Posture Active
AWS Security Hub, AWS Inspector, Defender for Cloud all enabled. Geo-blocking active on M365 and Sonicwall firewall. Containers run on least privilege. AWS CloudWatch deployed. Azure Monitor in progress.
Implemented · High
Annual Third-Party Pen Testing
External penetration tests conducted annually + post-change. Criticals remediated within 30 days, highs within 60. TPRM via UpGuard entering final review. IPS enabled on Sonicwall. Proactive security posture validated externally.
BAD
03 · Material Gaps

The Bad

These are known, documented gaps from the maturity self-assessment. Each represents a class of attack that Acclime currently has no mitigation for. Budget has been allocated to most — the risk is in delay.

Not Implemented · Critical
No SASE / ZTNA / DLP
No Zero Trust Network Access. No Data Loss Prevention. Roaming users access the web without a secure gateway. S$405K budgeted; feasibility completed by Andi. Decision must happen now. Every week of delay is a week of uncontrolled data egress.
Not Implemented · Critical
No PAM Solution
Privileged Access Management not implemented. PIM used as interim only. Admin passwords are not auto-rotated. Local admin accounts unmanaged. Service accounts share credentials. Risk: insider threat and attacker lateral movement post-compromise.
Not Implemented · Critical
No WAF on Applications
Web Application Firewall absent from Agility, Acuity and Aura. Group-IB explicitly models Acclime as HIGH risk without WAF. External attack surface is significantly elevated. Feasibility study in progress — accelerate immediately.
Not Implemented · Critical
Admin Accounts Not Segmented
Admin and everyday user accounts share the same identity. No separation between day-to-day and privileged access. Core Level 2 control. Dramatically increases blast radius of any account compromise — one phish grants admin access.
Not Implemented · Medium
On-Premise Servers Unencrypted
Windows on-premise servers lack server-storage encryption. BitLocker deployed on endpoints only — servers excluded. Physical access or server-level compromise exposes data in plaintext. Particularly relevant for offices with local infrastructure.
Partial · High
MS Secure Score: 59% (Target 80%+)
Microsoft's own security assessment rates Acclime at 59/100. Target is 80%+. Known gaps: legacy auth protocols, conditional access coverage, configuration hygiene. Progress has been made but score hasn't moved. Quarterly improvement programme needed.
